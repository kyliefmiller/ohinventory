Contribute

This is an ongoing project and we love any help we can get to make the database as complete and accurate as possible. Click on the options below to see more about how you can help!

We would love to include your oral histories!

Please send an email to colorado150southwest@gmail.com with the name and contact information for your institution and the following information about each oral history. If you don't have all of it, that is fine, but the more information the better!

  • The medium of the file (audio, video, and/or transcript)
  • The name of the narrator
  • The name of the creator
  • The date the oral history was created
  • A list of any locations discussed in the oral history, including counties
  • A brief description of topics discussed. If desired, you may look at the topic tags created (see the list in the database search) and list any matches.
  • Any other information you think is relevant!
